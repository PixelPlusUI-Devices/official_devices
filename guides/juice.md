# Installation Guide For PixelPlusUI on POCO M3/Redmi 9T/Redmi 9 Power/Redmi Note 9 4G (juice)

### CLEAN FLASH
1. Boot to Recovery
2. Wipe Dalvik Cache, Cache, Data
3. Format Data
4. Reboot to Recovery
5. Flash Vendor MIUI 12.5
6. Reboot Recovery
7. Flash PixelPlusUI
8. Flash DM-Verity
9. Reboot to System

### DIRTY FLASH
1. Boot to Recovery
2. Wipe Dalvik Cache/Cache
3. Flash PixelPlusUI
4. Reboot to System
